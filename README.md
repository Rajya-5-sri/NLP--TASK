A Jupyter Notebook‑based NLP pipeline was implemented on a 100+ word custom text about natural language processing using Python and standard NLP libraries. The workflow covers text preprocessing steps including stopword removal, punctuation cleaning, stemming, and lemmatization to normalize the raw text. Feature engineering techniques such as Bag of Words and TF‑IDF were applied to convert the text into numerical representations suitable for downstream tasks. Additionally, a Word2Vec model was trained on the custom corpus to learn dense word embeddings, enabling word similarity checks and exploration of semantic relationships between key terms in the text.
